## ⚙️ Installation
   Clone the repository and install the required packages:
   ```bash
   git clone https://github.com/playoung2818/Fun.git
   cd Fun
```
  ```bash
pip install -r requirements.txt

```
---
## Run
 ```bash
powershell -ExecutionPolicy Bypass -File .\starter.ps1
```
